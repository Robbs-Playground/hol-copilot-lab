# hol-copilot-lab

This repository is a hands-on lab for learning GitHub Copilot workflows. It includes:

- `eCommApp/`: The “Daily Harvest” sample e-commerce app (React + TypeScript + Vite)
- `Instructions/`: Step-by-step lab guides
- `AdditionalLearning/`: Optional supporting material

## Quick start (run the app)

**Requirements:** Node.js current LTS recommended (Node 20+).

1. Open a terminal and go to the app folder:
	```bash
	cd eCommApp
	```

2. Install dependencies:
	```bash
	npm install
	```

3. Start the dev server:
	```bash
	npm run dev
	```

For more details (build, tests, Windows notes), see `eCommApp/README.md`.
